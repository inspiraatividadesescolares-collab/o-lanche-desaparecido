# o-lanche-desaparecido
Escape room matemático, para crianças do 3º ano do Ensino fundamental
PASTA DO JOGO - O LANCHE DESAPARECIDO

VERSÃO AJUSTADA:
- 30 questões
- 6 níveis
- 5 questões por nível
- APENAS 1 PISTA POR NÍVEL

COMO AS PISTAS FUNCIONAM:
- A pista do nível aparece somente ao acertar a 5ª questão daquele nível.

Estrutura:
- index.html
- style.css
- script.js
- /img
- /sons

COMO USAR:
1. Abra o arquivo index.html no navegador.
2. Coloque suas imagens na pasta img.
3. Coloque seus áudios na pasta sons.
4. Substitua os textos [INSERIR IMAGEM ... AQUI] pelas imagens desejadas.
5. Nos pontos indicados no script.js, adicione os sons.

NOVA ALTERAÇÃO:
- A narrativa inicial foi dividida em 6 telas.
- Cada tela tem botão para avançar e, a partir da tela 2, também botão para voltar.

- As 6 telas da narrativa agora têm visual cinematográfico, com aparência de cenas de abertura.
- Foram adicionados elementos como legenda de cena, etiqueta de cena, contraste mais dramático e animações suaves.

- Esta versão foi ajustada para celular em posição de paisagem.
- A proposta é manter tudo 100% dentro da tela, sem rolagem.
- Foram reduzidos textos, botões, cartões e espaçamentos no mobile horizontal.
- O jogo abre ocupando a tela inteira do celular em modo paisagem.
